# Industrial Website

A responsive **industrial/company profile website** built with **HTML, CSS, and JavaScript**.  
Deployed on **Vercel** and hosted on **GitHub**.

## Live Demo
- Deployed URL: `https://industral-website.vercel.app/index.html`

> Tip: You can also open the homepage without `index.html` if clean URLs are enabled.

---

## Features
- Multi-page static website
- Responsive layout (works on mobile + desktop)
- Organized assets:
  - `css/` for styles
  - `js/` for scripts
  - `images/` for images (including subfolders)
  - `fonts/` for fonts
- Pages included (example):
  - Home (`index.html`)
  - About (`about.html`)
  - Services (`service.html`)
  - Team (`team.html`)
  - Gallery (`gallery.html`)
  - Contact (`contact.html`)
  - History (`history.html`)
  - Story (`story.html`)
  - Partner (`partner.html`)

## Run Locally
You can run it locally in any of these ways:

### Option 1: Open directly
- Double click `index.html`

### Option 2: VS Code Live Server (recommended)
1. Install **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

---

## Deploy on Vercel (from GitHub)
1. Push the project to a GitHub repository
2. On Vercel:
   - **Add New → Project**
   - Import your GitHub repo
   - Framework preset: **Other**
   - Build command: *(none)*
   - Output directory: *(none)*
3. Deploy
